Based on the data table to find the price and route.

Require packages

  • magrittr : For pipe flow (%>%) control
  • knitr : For beautiful display in R Markdown.
  • kableExtra : For beautiful display in R Markdown.
  • rgdal : To load shp in R.
  • GISTools : GIS finction package.

Build Relative Functions

  1. Create Data Matrix for Test

    I use the r function named rnorm to random create the value to be the price of traffic tickets. I make the list of price which created by rnorm function to be a matrix and rename the col and row to show the station (with code).
    In defult, the rnorm will randomly create values from normal distribution with mean value = 0 and standard deviation (sd) = 1. You can change the value when you use the function in R.

    The document of rnorm is here
  1. I give a Origin station and a Disitnation to find the ticket price.

    In this function, I use the which function to find the index with given value in the price matrix.
    The target is that I give the name of the origin station and distination and then R tells to me the price from the origin to distination.
  1. I give a specific price and find the all combinations of routes.
  1. I have a Origin Station with a specific price and want to know how many and which stations I can arrive.
